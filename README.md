# Topic Discussions

This repository hosts **topic discussion activities for pharmacy trainees**. These activities use interactive web-based tools such as quizzes and games to facilitate learning and discussion.

**Disclaimer:** The HTML code and questions are AI-generated. They are intended for educational purposes only and should be used **in conjunction with clinical expertise**.

## Deployment

Each activity is deployed via **GitHub Pages** and can be accessed using its HTML file name.

**Examples (direct links also provided):**

- `spectrum.html` → [https://shemtsai.github.io/topic-discussions/spectrum](https://shemtsai.github.io/topic-discussions/spectrum)  
- `Staph_aureus_bacteremia.html` → [https://shemtsai.github.io/topic-discussions/Staph_aureus_bacteremia](https://shemtsai.github.io/topic-discussions/Staph_aureus_bacteremia)
- `ampc_esbl.html` → [https://shemtsai.github.io/topic-discussions/ampc_esbl](https://shemtsai.github.io/topic-discussions/ampc_esbl)
- `CMV.html` → [https://shemtsai.github.io/topic-discussions/CMV](https://shemtsai.github.io/topic-discussions/CMV)
- `basic_diagnostics.html` → [https://shemtsai.github.io/topic-discussions/basic_diagnostics](https://shemtsai.github.io/topic-discussions/basic_diagnostics)
  
## How to Use

1. Open the activity in your browser using the deployed GitHub Pages link.  
2. Follow on-screen instructions to participate in quizzes or games.  
3. Use the content as a **discussion starter** or learning tool with trainees.
